# Stock Order Manager - Mobile Responsive (9:16)

Simple single-file static web app (HTML/CSS/JS) for managing stock orders on mobile screens (9:16 ratio).

## Features
- Add product + supplier/order reference
- Mark item as Done or Cancel
- Import items from JSON
- WhatsApp-style PDF preview UI
- Mobile-focused design (max-width: 420px)

## How to use
1. Open `index.html` in a browser (or host it on GitHub Pages).
2. Add items using the form.
3. To import items paste JSON like:
{"product":"Charger Cable","order":"Supplier A / PO-123"}

## Deploy on GitHub
1. Create a new repo on GitHub.
2. Upload the files from this folder (`index.html`, `README.md`).
